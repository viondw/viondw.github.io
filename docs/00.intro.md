# 引导页

## 前言

 嗨，看来你已经可以自己找到这个文档了，左边的侧边栏也会对你有所帮助，尝试浏览一下吧。



 
 ### 😊文档正在编写中，欢迎提交你的文档/代码/页面美化。😊


 

 ## 如何提交/修改页面文档

- 🚧请注意，帖子的保存格式应为.md格式，例如Q&A.md。🚧
- 🚧严格遵循markdown语法，换行不需要额外语法。🚧
 
- 你可以在每个页面的左下角看到**编辑此页**，点进去右上角进行修改，标题可以告诉我你做了什么。😇

### 提交新文档

- 在**docs**中的文件夹对应侧边栏的分类，帖子放置在对应的文件夹内，如直接放置在**docs**中则会自动出现在侧边栏作为一个独立页面。

 ## 修改站点信息/美化
- 网站的原信息存储在**docusaurus.config.ts**中，首页在**src/pages**。😋
